## 🏏 IPL Data Analysis

A comprehensive exploratory data analysis (EDA) of the Indian Premier League (IPL),
covering match statistics, player performances, team comparisons, and seasonal trends
from 2008 to 2024.

### 📌 Overview
The Indian Premier League (IPL) is the world's most popular T20 cricket league.
This project leverages historical IPL datasets to extract meaningful insights and
visualize patterns that drive match outcomes and player success.

### 🔍 Key Analysis Areas
- **Match Analysis** — Win/loss trends, toss impact, venue performance
- **Batting Statistics** — Top run-scorers, strike rates, boundary counts
- **Bowling Statistics** — Economy rates, wicket takers, dot ball analysis
- **Team Performance** — Season-wise rankings, head-to-head comparisons
- **Player Insights** — Consistent performers, MVP trends across seasons

### 🛠️ Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- Jupyter Notebook
- Dataset: Kaggle IPL Dataset (2008–2024)

### 📊 Visualizations Include
- Bar charts, heatmaps, line graphs, pie charts
- Interactive dashboards (if applicable)
- Season-wise performance trends
